# stree

Just like the `tree` command, but simpler

## building

`sudo make install`

## Usage

```
    Usage: stree [options]
    Options:
    <path>                  Directory to display
    -h                      Show help menu
    -d <num>                Folder depth
    -r <num>                Amount of roots
    -i <num>                Ignore files (max. 10)
```
